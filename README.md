# Telegram WhatsApp Messaging Bot

A Telegram bot to send messages to numbers in a VCF file via multiple WhatsApp accounts using baileys.

## Features
- /pair: Link WhatsApp accounts
- /send: Send messages to numbers in VCF
- Multi-account support

## Setup
1. Clone repo & install dependencies
2. Add your Telegram bot token to `.env`
3. Run with `npm start`

## Deploy
- Push to GitHub
- Host on Render
